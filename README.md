# Car Showroom Website

A responsive landing page for a car showroom built using **HTML**, **CSS**, and **Vanilla JavaScript**. The project showcases a modern layout with interactive components designed to provide a clean browsing experience for visitors.

## Features

- Responsive design
- Modern landing page layout
- Navigation bar with smooth section navigation
- Image slider for featured vehicles
- Feature cards
- Pricing plans
- Vehicle price table
- Frequently Asked Questions (FAQ)
- Contact form
- Interactive UI with JavaScript

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/car-showroom-web.git
```

Open the project folder and launch `index.html` in your browser.

No additional installation or dependencies are required.
